Certificados

WHALE CERTIFICADOS é um projeto opensource capaz de automatizar a geração de certificados. Basta inserir um layout com campos sem preenchimento e um arquivo csv com os nomes a serem gravados.




Cores

WHALE CORES é um projeto opensource capaz de remover o fundo com base na presença de uma determinada cor na imagem. Basta inserir uma imagem e digitar o valor hexadecimal da cor alvo; automaticamente o WHALE marcará como transparente.




Códigos

WHALE CÓDIGOS é um projeto capaz de gerar códigos QR-Code de qualquer link HTTP ou HTTPS, fazendo download automático em formato PNG




Segurança

WHALE SEGURANÇA é um projeto capaz de criptografar e descriptografar seus arquivos, através de uma chave / senha personalizada
